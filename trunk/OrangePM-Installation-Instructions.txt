1: Get the checkout (svn co https://orangepm.googlecode.com/svn/trunk [folder name]).
2. Set proper permissions (sudo chmod -R 777 [folder name]).
3. Build the model(php symfony doctrine:build-model).
4. Create config/databases.yml (php symfony configure:database "mysql:host=localhost;dbname=[DB name]" [DB username] [DB password]).
5. Build SQL (php symfony doctrine:build-sql).
6. Insert SQL into the database (php symfony doctrine:insert-sql).
7. Load initial data (php symfony doctrine:data-load).
